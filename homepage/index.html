常见问题

关于 autoWeChat 的常见疑问

通过 macOS 的 AppleScript 或 Windows 的 pyautogui 自动化操控微信客户端，模拟键盘操作完成搜索联系人、粘贴消息、发送。无需微信 API，安全可靠。

需要在电脑上安装微信客户端并保持登录状态。macOS 支持微信、企业微信等多个实例。

支持 macOS 12.0 及以上版本，以及 Windows 10/11 64 位系统。

所有数据存储在本地，不会上传到第三方服务器。AI 调用仅发送课程概要，不包含学员和家长的个人信息。

接入 DeepSeek AI，根据您填写的课程概要自动扩展为 200 字左右的完整课程汇报内容，同时可根据学员表现概要生成个性化评语。
